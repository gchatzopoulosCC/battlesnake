(static) avoidWalls(gameState, isMoveSafe) → {Object}

This function checks if the snake's head is at the edge of the game board and updates the isMoveSafe object to prevent the snake from moving off the board. Hitting the boundary of the game board results in an immediate game over.
Parameters:
Name Type Description
gameState Object The current state of the game.
Properties
Name Type Description

Properties

Properties
Name Type Description
body Array.<Object> An array of objects representing the snake's body segments.
isMoveSafe Object An object representing the safety of each move.
Properties
Name Type Description

right boolean Indicates if moving right is safe.
up boolean Indicates if moving up is safe.
down boolean Indicates if moving down is safe.

Example
const gameState = {
  board: {
    width: 11,
    height: 11,
  },
  you: {
    body: [
      { x: 0, y: 5 },  // Head at left edge of board
      { x: 1, y: 5 },
      { x: 2, y: 5 },
    ],
  },
};

const isMoveSafe = {
  left: true,
  right: true,
  up: true,
  down: true,
};

avoidWalls(gameState, isMoveSafe);
// After calling this function:
// isMoveSafe = {
//   left: false,  // Moving left would hit the wall
//   right: true,
//   up: true,
//   down: true,
// }
