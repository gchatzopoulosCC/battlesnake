Module: src/helper/sets/collisionSet

This file contains a function to create a Set of positions occupied by other snakes on the board. This Set is used for efficient collision detection when evaluating potential moves during gameplay.

Methods

(static) getCollisionSet(gameState) → {Set.<string>}

Creates a Set containing string representations of all segments of other snakes on the board. This is used to check if a move would result in a collision with another snake, which would end the game.
Parameters:

gameState Object The current state of the game.

Example
const gameState = {
  board: {
    snakes: [
      {
        id: "snake-1",
        body: [
          { x: 5, y: 5 },
          { x: 5, y: 6 }
        ]
      },
      {
        id: "snake-2",
        body: [
          { x: 7, y: 8 },
          { x: 7, y: 7 }
        ]
      }
    ]
  },
  you: {
    id: "snake-1"
  }
};

const collisionSet = getCollisionSet(gameState);
// Returns a Set containing: "7,8", "7,7"

// The Set can be used to check if a move would result in a collision:
collisionSet.has("7,8"); // true
collisionSet.has("5,5"); // false (player's snake positions are excluded)
