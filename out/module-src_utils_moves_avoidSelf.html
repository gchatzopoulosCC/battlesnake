Module: src/utils/moves/avoidSelf

This file contains a function to check for collisions with the snake's own body and update the isMoveSafe object accordingly. Avoiding self-collisions is critical for survival as running into your own body ends the game immediately.

(static) avoidSelf(gameState, isMoveSafe) → {Object}

This function checks for potential collisions with the snake's own body and updates the isMoveSafe object accordingly to avoid self-collisions. It uses a specialized avoidance checker that works with a set of the snake's body segments (excluding the head).
Parameters:

gameState Object The current state of the game.

Name Type Description

Properties
Name Type Description

isMoveSafe Object An object representing the safety of each move.
Properties
Name Type Description

down boolean Indicates if moving down is safe.

Example
const gameState = {
  you: {
    body: [
      { x: 5, y: 5 },  // Head
      { x: 5, y: 4 },
      { x: 6, y: 4 },
      { x: 6, y: 5 }
    ]
  }
};

const isMoveSafe = {
  left: true,
  right: true,
  up: true,
  down: true,
};

avoidSelf(gameState, isMoveSafe);
// After calling this function:
// isMoveSafe = {
//   left: true,
//   right: true,
//   up: true,
//   down: false,  // Moving down would collide with body at "5,4"
// }
