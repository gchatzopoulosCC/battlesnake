Module: src/core/snake

This file contains the main logic for the Battlesnake's movement decisions. It analyzes the game state and determines the next move by avoiding dangerous positions.

Methods

(inner) move(gameState) → {Object}

Determines the next move for the Battlesnake based on the current game state. The function evaluates possible moves and avoids unsafe options such as walls, the snake's own body, and other snakes. If no safe moves are available, it defaults to moving "down".
Parameters:
Name Type Description
gameState Object The current state of the game.
Properties
Name Type Description
board Object The board object containing the game board's dimensions, food, and snakes.
Properties
Name Type Description
width number The width of the game board.
height number The height of the game board.
food Array.<Object> An array of food positions, each represented as an object with `x` and `y` coordinates.
snakes Array.<Object> An array of snakes on the board, including their positions and properties.
turn number The current turn number of the game.
you Object The player's snake object.
Properties
Name Type Description
id string The unique identifier for the player's snake.
body Array.<Object> An array of objects representing the snake's body segments, starting with the head.
health number The current health of the player's snake.
head Object The position of the snake's head, represented as an object with `x` and `y` coordinates.
tail Object The position of the snake's tail, represented as an object with `x` and `y` coordinates.

Example
const gameState = {
  board: {
    width: 11,
    height: 11,
    food: [{ x: 5, y: 5 }],
    snakes: [{ id: "1", body: [{ x: 1, y: 1 }, { x: 1, y: 2 }] }]
  },
  turn: 1,
  you: {
    id: "1",
    body: [{ x: 1, y: 1 }, { x: 1, y: 2 }],
    health: 100,
    length: 2,
    head: { x: 1, y: 1 },
    tail: { x: 1, y: 2 }
  }
};

move(gameState);
// Returns { move: "right" } or another safe direction

// If no safe moves are available:
// Returns { move: "down" }
