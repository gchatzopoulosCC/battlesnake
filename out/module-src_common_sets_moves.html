Methods

(static) getMoves(gameState) → {Array.<Object>}

Retrieves all possible moves for the snake based on its current head position. Each move includes the direction and the adjacent position it leads to.
Parameters:
Name Type Description
gameState Object The current state of the game.
Properties
Name Type Description
you Object The player's snake object.
Properties
Name Type Description
body Array.<Object> An array of objects representing the snake's body segments, starting with the head.
Source:
  • helper/sets/moves.js, line 42
Returns:
- An array of possible moves, where each move is an object containing: - `direction` {string}: The direction of the move (`"left"`, `"right"`, `"up"`, `"down"`). - `adjacentPosition` {string}: The adjacent position in string format (`"x,y"`).
Type
Array.<Object>
Example
const gameState = {
  you: {
    body: [
      { x: 5, y: 5 },
      { x: 5, y: 4 },
    ],
  },
};
getMoves(gameState);
// Returns:
// [
//   { direction: "left", adjacentPosition: "4,5" },
//   { direction: "right", adjacentPosition: "6,5" },
//   { direction: "up", adjacentPosition: "5,6" },
//   { direction: "down", adjacentPosition: "5,4" },
// ]
