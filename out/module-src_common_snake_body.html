Module: src/common/snake/body

This file contains utility functions to retrieve the head and tail segments of a snake. These functions simplify access to the snake's body parts and are commonly used in movement logic.

(inner) getHead(gameState) → {Object}

Returns the head (first segment) of the snake from the game state. The head is used for determining the snake's current position and calculating possible moves.
Parameters:
Name Type Description
gameState Object The current state of the game.
Properties
Name Type Description
you Object The player's snake object.
Properties
Name Type Description
body Array.<Object> An array of objects representing the snake's body segments, starting with the head.
Source:

Returns:
- The head of the snake, represented as an object with `x` and `y` coordinates.
Type
Object
Example
const gameState = {
  you: {
    body: [
      { x: 1, y: 1 },  // Head
      { x: 1, y: 2 },
    ],
  },
};
getHead(gameState); // Returns { x: 1, y: 1 }

(inner) getTail(gameState) → {Array.<Object>}

Returns all body segments of the snake excluding the head. The tail segments are used for collision detection to prevent self-collisions.
